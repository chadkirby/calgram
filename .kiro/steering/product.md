# Product Overview

## CalGram - Calorie Tracker

A comprehensive calorie tracking application built with Jazz (real-time collaborative framework) that enables users to log meals, track weight changes, and visualize their health data over time.

### Core Features

- **Meal Logging**: Track food intake with detailed nutritional information (calories per gram, weight, categories)
- **Weight Tracking**: Record weight measurements with timestamps and notes
- **Daily Summaries**: View daily calorie consumption broken down by food categories with pie charts
- **Trend Analysis**: Visualize long-term patterns in calorie intake and weight changes with LOWESS-smoothed trend lines
- **Food Intelligence**: Auto-complete and remember frequently used foods with their nutritional data

### Key Value Propositions

- Real-time data synchronization across devices using Jazz framework
- Intelligent food suggestions based on usage history
- Visual analytics for understanding eating patterns and weight trends
- Offline-capable with automatic sync when reconnected
- Passkey authentication for secure, passwordless access

### Target Users

Individuals focused on health and fitness who want to track their calorie intake and weight changes with detailed analytics and cross-device synchronization.
